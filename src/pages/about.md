The Global Image Data Ecosystem (GIDE) connects different biological and preclinical image data resources across the globe to allow metadata and data sharing. You can read more about the GIDE project [here](https://founding-gide.eurobioimaging.eu/).

This search portal provides a single federated search interface over a number of biological imaging databases that contribute to gide. We currently display data from the [Image Data Resource (IDR)](https://idr.openmicroscopy.org/), [SSBD:repository](https://ssbd.riken.jp/repository/) and [SSBD:database](https://ssbd.riken.jp/database/), and the [BioImage Archive (BIA)](https://www.ebi.ac.uk/bioimage-archive/).
